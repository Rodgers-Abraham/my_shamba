# my_shamba

A cross-platform (Mobile & Web) farm management application built with Clean Architecture, Firebase, and BLoC.

## Architecture
- **Domain**: Entities, Repository Interfaces, Use Cases.
- **Data**: Models, Repository Implementations, Data Sources (Firestore/Local Cache).
- **Presentation**: BLoC State Management, Responsive UI Widgets/Screens.

## Tech Stack
- Flutter
- Firebase (Auth, Firestore, Messaging)
- BLoC
- Isar (Local Persistence)
- flutter_local_notifications
